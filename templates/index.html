About the Project

What This Project Is About

A comprehensive data analytics journey from raw data to insightful dashboards

Objective

To study heart disease-related data and identify key risk patterns across gender, age, lifestyle, and health conditions — then present findings through clean, interactive Tableau dashboards.

Dataset

The CDC BRFSS 2020 dataset with 319,000+ survey responses covering 14 health indicators including BMI, smoking, alcohol, physical activity, diabetes, stroke, and more.

MySQL Data Cleaning

Raw data was imported into MySQL, duplicate records removed with DISTINCT, filtered using WHERE, aggregated with GROUP BY, and organized into analytical views.

Tableau Visualizations

The cleaned dataset powers 10 interactive worksheets, 3 comprehensive dashboards, and 1 guided story on Tableau Public for clear, compelling storytelling.

Flask Web Integration

A Python Flask backend serves this responsive web interface, embedding live Tableau Public visualizations for a seamless, interactive analytics experience.
